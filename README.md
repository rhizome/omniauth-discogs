# OmniAuth::Discogs

OmniAuth strategy for Discogs (http://www.discogs.com/)

## Installation

Add this line to your application's Gemfile:

    gem 'omniauth-discogs'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-discogs
